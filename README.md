# Анализ лояльности пользователей Яндекс Афиши с помощью Python

## Цель проекта

Провести исследовательский анализ данных для команды маркетинга, чтобы лучше понимать поведение пользователей, и изучить информацию о том, какие пользователи с большей вероятностью возвращаются на платформу и делают заказы повторно.

## Задачи

- Быстро выявлять перспективных клиентов и предлагать им персонализированные условия.
- Точно настраивать рекламу на аудитории с высокой вероятностью возврата.
- Оптимизировать маркетинговые бюджеты.
- Повысить общий уровень удержания клиентов.

## Используемые библиотеки

- pandas
- numpy
- matplotlib
- seaborn
- sqlalchemy
- python-dotenv

## Выводы

- Основная доля клиентов делает единичные заказы — рекомендуется внедрить промокоды для стимулирования повторных покупок.
- Самые лояльные клиенты покупают 2-3 билета со средним чеком около 500 руб. — для них стоит настроить персональные предложения и систему рекомендаций.
- Обнаружен пользователь с аномально большим количеством заказов — необходимо проверить данные и при необходимости исключить из анализа.
